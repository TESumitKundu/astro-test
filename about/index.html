About This Site

Astro Git CMS Blog is a simple content website built around static generation and Git-managed content.

Editors can update blog posts, page copy, settings, and images through Pages CMS. Developers can review every content change in Git, and hosting platforms can rebuild the site automatically after each commit.

This approach is ideal for teams that want a production-ready website without managing WordPress, a database, or a server-side CMS.
